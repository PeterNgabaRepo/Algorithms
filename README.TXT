Repository should include:
graph.h - containing vector, edge and graph classes
minHeap.h - containing heap class
minHeap.cpp - defintion of heap fxns
driver.cpp  - combines everything and generates solution
	      using user inputted filename for the data and index for the root vertex

makefile - containing the code to compile the file
test.txt - sample data file to use or augment

Steps to run:
Works on @GL.UMBC.EDU servers
type "make driver" to compile
type "./driver" to execute
type "test.txt" to select that file or another file that has data in the same format
input a valid index of the root vertex you decide to use
